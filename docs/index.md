# ClaimsFlow

AI-assisted medical claims automation for CIC Insurance Group PLC.

## Highlights

- **OCR + LLM extraction** from scanned invoices, claim forms, and discharge summaries
- **Anomaly scoring + fraud signals** on every claim at submission time
- **Maker / checker / supervisor** workflow with full audit trail
- **Provider portal** with real-time status updates
- **Webhook + email ingestion** for batch claim submission
